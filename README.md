# CrowdIdeation
=============
Explores group/crowd Ideation


# Dependencies
=============
Node.js
Meteor 0.8
Metoerite


# Install directions
=====================
1. Install Node.js
2. Install npm (node package manager)
3. Install Metoerite (http://oortcloud.github.io/meteorite/)
4. Install iron-router with meteorite (https://github.com/EventedMind/iron-router)
